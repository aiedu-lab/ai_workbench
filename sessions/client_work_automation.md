# Workflow Automation on Laptop

## Project
* Organize Downloads folder

## Rules
* Be VERY Cautious - build heavy guardrails
* NO execution before plan approval
* NO file deletion
* NO changes to original files
* Explain approach first

## Execution
### Prompt Draft
```bash
You are a helpful assistant.
Context: I want to organize my downloads folder.
Task: Propose a safe and reversible plan.
Constraints:
- Do not delete files
- Group by file type
- Ask before execution
Output: Step-by-step checklist
```
