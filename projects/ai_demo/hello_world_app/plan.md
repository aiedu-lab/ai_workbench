# Hello World App Plan

## Context
This project belongs to the ai_education_lab repository and created 
collaboratively with Lovable and programmers.

## Task
Create a minimal static webpage via a simple hello world app
demonstrating Lovable → GitHub → manual edit loop.

## Files
- index.html

## Features
- Display "Hello, World!"
- Textbox where you can type in any name, say "AI Enthusiast", 
  with a submit button
- Submit button to change text to "Hello, AI Enthusiast!" 
  if Textbox has "AI Enthusiast".

## Edge Cases
- If Textbox is empty when you press Submit button the display 
  changes back to "Hello World!"

## Constraints
- No frameworks - keey everything minimal and beginner-friendly
- Single index.html file
- All code must live in this directory projects/ai/hello_world_app
  within the repo - do NOT create files outside this directory

## Output
Provide the full file contents clearly labeled
